프로젝트 발표: "영화는 왜 성공할까? 캐글 영화 데이터로 보는 평점과 흥행의 관계"
1. 프로젝트 개요
캐글 영화 데이터를 활용한 분석 프로젝트

목표: 영화의 평점과 시대적 흐름의 관계를 탐색하고, 간단한 예측 모델 실험

"흥행 성공 영화에는 어떤 특성이 있을까?"라는 호기심에서 시작

2. 데이터셋 소개
Kaggle: The Movies Dataset (TMDB 기준)

주요 컬럼:

vote_average: 평점

revenue, budget: 수익, 제작비

genres, release_date, runtime 등

총 약 100만개 영화 데이터

3. 분석 방향 설정
‘영화의 성공’을 어떻게 정의할까?

평점이 7점 이상일 경우 평범
평점이 7.5점 이상일 경우 성공
평점이 8.0점 이상일 경우 대성공

투표수 문제로 컷오프 :
투표수가 낮음에도 평점이 높은 경우가 있기때문에
4표 이상의 영화로 컷오프
 - 대부분의 영화들이 0~4표(90퍼센트)이기 때문에 유효한 데이터를 얻기 위해 4표로 컷오프

4. 데이터 전처리 개요
genres 컬럼: JSON 또는 문자열 형태를 파싱하여 리스트 형태로 변환

드라마(Drama)와 다큐멘터리(Documentary)는 단일 장르일 때만 유지
→ 다른 장르와 결합 시 제외

vote_average, vote_count, runtime 등 주요 수치형 컬럼은 숫자형으로 변환

결측치 및 이상치 확인

5. EDA 1 – 평점 분포
평균 평점: 약 6.1

평점 분포 시각화 (히스토그램, 박스플롯)

투표수(vote_count) 분포 확인

컷오프 적용: 투표수 ≥ 4인 영화만 필터링

90% 이상의 영화가 투표수가 0~4표에 집중되어 있어 데이터 품질 문제

컷오프 적용 후 평점 분포 재확인

6. EDA 2 – 평점 등급 구간화
평점 구간 정의:

7.0 ~ 7.4 : 평범

7.5 ~ 7.9 : 성공

≥ 8.0 : 대성공

각 구간별 영화 개수, 비율 시각화 (막대그래프)

평점 구간별 장르 분포 확인

7. EDA 3 – 장르별 분석
장르별 평균 평점 계산

평점 구간(평범/성공/대성공)에서 장르별 비율 확인

예:

드라마/다큐멘터리는 평균 평점이 높지만, 투표수 컷오프 후 변동 여부 확인

액션/어드벤처 장르는 평균 평점이 보통 수준이나 대중성↑

8. 부가 실험 – 평점 vs 수익
ROI 계산: revenue / budget (단, 두 값 모두 0이 아닌 영화만)

ROI > 1이면 ‘성공’으로 간주

ROI와 평점 구간의 관계 확인

데이터 손실 및 한계점 명시

9. 변수 선택 및 가공
수치형 변수: vote_average, vote_count, runtime

범주형 변수: genres (One-hot encoding)

타겟 변수:

평점 기반 등급(3단계)

ROI 기반 성공 여부(부가 실험용)

10. 모델 선정 및 설명
간단한 분류 모델 2종:

로지스틱 회귀 (해석 용이)

랜덤포레스트 (비선형 패턴 포착 가능)

평점 기반 타겟으로 모델 학습

ROI 타겟은 데이터 수 적음 → 실험 수준

11. 모델 학습 및 평가 방법
데이터 분할: Train/Test = 80/20

평가 지표:

정확도(Accuracy)

정밀도(Precision)

재현율(Recall)

교차 검증은 단순 언급

12. 로지스틱 회귀 결과 (평점 타겟)
정확도: 약 70%

중요한 변수: vote_count, runtime, 일부 장르

모델 계수 시각화 (bar chart)

13. 랜덤포레스트 결과 (평점 타겟)
정확도: 약 73%

중요 변수: vote_count, runtime, 특정 장르

depth 제한으로 과적합 방지

14. 중요 변수 시각화
변수 중요도 bar plot

분석 결과 요약:

투표수와 일부 장르가 평점 등급 예측에 영향

평점 구간은 장르 특성과 결합 시 흥미로운 패턴 발견

15. 시각화 정리
주요 시각화 한 장에 모음:

평점 분포(컷오프 전/후)

평점 구간별 영화 수

장르별 평균 평점

모델 중요도 그래프

16. 프로젝트에서 배운 점
투표수 컷오프의 필요성을 실감

평점 데이터만으로도 다양한 인사이트 가능

ROI 분석 시 데이터 품질이 중요하다는 점 인지

17. 한계점
평점은 개봉 후 형성되는 지표 → 사전 예측에는 제약

ROI 분석 데이터가 부족

외부 요인(홍보, 배우, 시즌 등) 미반영

18. 추가 아이디어
리뷰 텍스트 감성 분석

감독/배우의 과거 흥행 기록 반영

개봉 시기(성수기/비수기) 변수화

19. 결론
평점 기반 분석은 장르와 투표수의 관계에서 의미 있는 패턴 발견 가능

ROI와의 관계는 데이터 품질 이슈로 제한적
