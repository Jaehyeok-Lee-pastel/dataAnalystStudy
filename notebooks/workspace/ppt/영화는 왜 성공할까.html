📊 로지스틱 회귀

✅ 선정 이유

뛰어난 해석력: 계수를 통한 직관적 이해
빠른 학습: 대용량 데이터 처리 가능
베이스라인: 기본 성능 확인에 적합
계수 분석: "액션 장르는 성공 확률을 20% 높인다"

⚙️ 작동 원리

점수 계산 → 시그모이드 변환 → 확률 → 분류

예) 액션(+2점) + 2020년대(+3점)
= 5점 → 85% 확률 → 성공

❌ 한계점

선형 관계만 학습: 복잡한 상호작용 못 잡음
특정 조합 효과: "액션+1980년대"만의 특별한 패턴 학습 어려움

🌲 랜덤포레스트

✅ 선정 이유

비선형 패턴: 장르×시대 조합 효과 학습
과적합 방지: 여러 트리의 앙상블 효과
변수 중요도: 어떤 특성이 중요한지 측정
안정성: 노이즈에 강함

⚙️ 작동 원리

여러 결정트리 → 각각 다른 관점으로 예측
→ 다수결 투표

트리1: "2020년대 + 뮤직 → 대성공"
트리2: "애니메이션 → 성공"
트리3: "액션 + 호러 → 평범"
→ 최종: 투표 결과로 결정

✅ 특장점

복잡한 규칙: "2010년대 다큐멘터리는 대성공하지만, 1980년대는 아니다"
상호작용: 장르와 시대의 조합 효과 자동 학습

🔍 모델 비교 요약

로지스틱 회귀: 해석 용이 + 빠른 실행 + 직관적 계수
랜덤포레스트: 복잡 패턴 학습 + 안정적 성능 + 변수 중요도
선택 기준: 해석력이 필요하면 로지스틱, 성능이 중요하면 랜덤포레스트
