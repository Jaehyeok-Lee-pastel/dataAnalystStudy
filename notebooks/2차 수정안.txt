프로젝트 발표: "영화는 왜 성공할까? – 평점과 시대 흐름으로 보는 성공 예측"
1. 프로젝트 개요
데이터: Kaggle TMDB Movies Dataset (약 100만 건)

목표:

영화의 평점과 시대별 패턴을 분석

이를 기반으로 향후 제작될 영화의 성공 가능성을 예측하는 모델 구현

배경:

원래는 수익(ROI)과 연계하려 했으나, revenue·budget 데이터 부실로
→ 평점과 시대 흐름 중심으로 분석·예측 방향 전환

2. 데이터셋 소개
출처: Kaggle TMDB Movies Dataset

주요 컬럼: vote_average, vote_count, genres, release_date, runtime 등

데이터 규모: 약 100만 건

3. 분석 방향 설정
성공 정의 (평점 기준)

6.5 ~ 7.4 → 평범

7.5 ~ 7.9 → 성공

≥ 8.0 → 대성공

데이터 신뢰성 확보:

1980~2019년: 투표 수 ≥ 4

2020~2025년: 투표 수 ≥ 1

시대 구분: 1980s ~ 2020s

4. 데이터 전처리 개요
genres 컬럼 JSON/문자열 파싱 → 리스트 변환

드라마·다큐멘터리: 단일 장르일 때만 유지

수치형 컬럼(vote_average, vote_count, runtime) 숫자 변환

결측치·이상치 처리

개봉 연도/시대(year, decade) 컬럼 생성

5. EDA 1 – 평점 분포
평균 평점 약 6.1

평점·투표수 분포 시각화

투표수 컷오프 적용 후 평점 분포 재확인
→ 90% 이상이 투표수 0~4표, 컷오프 필요성 확인

6. EDA 2 – 평점 구간화
구간 정의: 평범 / 성공 / 대성공

각 구간별 영화 개수·비율 시각화

장르별 평점 구간 비율 비교

7. EDA 3 – 장르별·시대별 분석
장르별 평균 평점, 평점 구간별 비율 확인

시대별 장르 변화 패턴 파악
(예: 1980s~2000s → 코미디·드라마 중심, 2010s 이후 → 다큐·뮤직 강세)

8. 변수 선택 및 가공
수치형: year (또는 decade)

범주형: 주요 장르(One-hot Encoding)

타겟: 영화 평점(vote_average)

예측 후 구간화하여 성공 여부 판단

필요 시 파생 변수:

시대별 장르 트렌드 지표(상승 +, 하락 -)

9. 모델 선정 및 설명 (장르+시대 기반 평점 예측)
목적

영화의 장르와 **개봉 시대(Decade)**를 기반으로 예상 평점을 예측

예측값을 구간화(평범/성공/대성공)하여 성공 가능성 판단

모델 선택

로지스틱 회귀: 변수 영향력 해석 용이

랜덤포레스트: 장르·시대 조합에서의 비선형 패턴 탐지

변수

장르(One-hot Encoding)

개봉 시대(Decade)

타겟

평점(vote_average) → 구간화

10. 모델 학습 및 평가 방법
데이터 분할: Train/Test = 80/20

평가 지표: Accuracy, Precision, Recall

교차검증은 보조 확인

11. 로지스틱 회귀 결과
정확도: 약 70%

주요 변수: 특정 장르·시대 조합

계수 분석을 통해 장르·시대별 긍/부정 영향 확인

12. 랜덤포레스트 결과
정확도: 약 73%

변수 중요도: 장르 > 시대

비선형 관계에서 장르와 시대 상호작용 포착

13. 중요 변수 시각화
로지스틱 회귀 계수 시각화

랜덤포레스트 변수 중요도 bar plot

14. 시각화 정리
평점 분포(컷오프 전/후)

평점 구간별 영화 수

장르별 평균 평점

모델 변수 중요도

15. 프로젝트에서 배운 점
투표수 컷오프의 중요성

장르·시대 정보만으로도 일정 수준의 평점 예측 가능

시대별 장르 흐름이 장기적인 성공 가능성 판단에 도움

16. 한계점
장르·시대만으로는 예측 정확도 한계

홍보·배우·제작사 등 외부 요인 미포함

평점은 개봉 후 형성되는 지표 → 완전 사전 예측은 제한적

17. 추가 아이디어
장르별 시대 트렌드 지표(상승/하락) 변수화

감독/배우 과거 성과 반영

리뷰 텍스트 감성 분석

개봉 시기(성수기·비수기) 변수화

18. 결론
장르와 시대 정보를 기반으로 한 평점 예측은 간단하지만 의미 있는 첫걸음

시대별 장르 트렌드 분석을 통해 제작 기획 시 참고 가능

추가 변수 반영 시 예측 성능 향상 기대
